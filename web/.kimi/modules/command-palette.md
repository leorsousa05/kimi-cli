# Módulo: Command Palette (`src/components/command-palette.tsx`)

## O que faz
Paleta de comandos acessível via Cmd/Ctrl+K.

## Ações disponíveis
- New Session
- Search Messages
- Toggle Theme
- Toggle Sidebar
- Pin/Unpin Session
- Keyboard Shortcuts

## Hook
- `useCommandPalette()`: gerencia estado open/close
